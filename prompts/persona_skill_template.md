# Persona Skill Generator Template

When the user completes a persona analysis, generate a self-contained SKILL.md
for that person using the structure below. Write it to `personas/{slug}/SKILL.md`.

This file IS the memory. No external database needed. Claude loads it and
immediately has full context — no re-entry of information required.

---

## How to Generate the File

Replace every `{placeholder}` with actual analyzed content.
The file must be fully self-contained — no references to external intake data.

---

## Active Instructions for Claude

When this skill is active, you have full context on {codename}.
Do NOT ask the user to re-describe this person. Use the profile above directly.

**For any query about {codename}:**
1. Use the personality model above to inform every response
2. Apply cultural overlay: {cultural_notes}
3. Always offer 2-3 variants calibrated to different strategic goals

**Quick command responses:**

`/pc prep` or `prep meeting`
→ Output a 6-line pre-meeting cheat sheet:
  1. Their current mood/context (if known)
  2. What they care about most right now
  3. One thing to say that will land well
  4. One thing to avoid
  5. Optimal opening line
  6. Your goal and how to frame it

`/pc tone` or `tone check`  
→ Output ONE line: the tone to use when messaging them right now.
   Example: "Concise and data-led. Skip pleasantries. Lead with the ask."

`/pc draft "[context]"`
→ Generate a ready-to-paste message. 3 variants:
   - Variant A: [optimizes for ___]
   - Variant B: [optimizes for ___]  
   - Variant C: [optimizes for ___]

`/pc simulate "[scenario]"`
→ Predict their reaction step by step, then give recommended strategy + script.
   Load scenario framework from persona-compass/references/scenario_templates.md

`/pc update`
→ Ask: "What did you observe?" Then append to observations.md and refine this model.
